# E-commerce REST API

A production-ready E-commerce Backend REST API built with Node.js, Express.js, and MongoDB.

## Project Goals

This project is being developed using modern backend development practices and is designed to support:

- React Customer Website
- React Admin Dashboard
- Future Mobile Applications

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Redis
- AWS S3
- Docker
- Swagger / OpenAPI
- Jest
- ESLint
- Prettier

## Current Status

🚧 Project initialization completed.

## Getting Started

### Install dependencies

```bash
npm install
```

### Create environment file

Copy:

```text
.env.example
```

to:

```text
.env
```

and provide the required values.

### Run the application

```bash
npm start
```
